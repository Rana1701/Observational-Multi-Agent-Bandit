# Observational-Multi-Agent-Bandit
A research project that studies the LLM's behaviour in an observational multi-agent Bandit environnement , whether the LLM can :
- Explore ,
- Detect bad agents ,
- Follow the good ones , 
- Keep a logarithmic regret O(ln n ) ,
- Learn in context , relying only on its rewards and history of actions
- ...etc

We try to tackle those questions by performing several experiments in this project .
